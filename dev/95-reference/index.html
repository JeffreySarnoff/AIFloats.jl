FloatsForML.CODE — Type
CODE

The built-in Unsigned Integer types available for encodings.

  • UInt8 for bitwidths <= 8
  • UInt16 for bitwidths > 8

FloatsForML.index_to_offset — Method
index_to_offset(x)

convert the Julia index x into a P3109 encoding value as a UInt16

  • >> (0x0100) ↦ 0x00ff
  • in 0-based languages, this is a do nothing operation
source
FloatsForML.offset_to_index — Method
offset_to_index(x)

convert the P3109 encoding x into a Julia index as an UInt16

  • >> (0xff) ↦ 0x0100
  • in 0-based languages, this is a do nothing operation
